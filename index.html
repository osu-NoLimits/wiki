    Welcome to osuNoLimits¶

    The complete development and deployment guide for building your own osu! private server

    What is osuNoLimits?¶

    osuNoLimits is a powerful, feature-rich osu! private server stack that combines modern web technologies with a robust game server backend. Built for developers and server administrators who want a production-ready solution with extensive customization options.

    Key Features¶

    Production Ready

    • Containerized Deployment — Simple, reproducible setup with Docker
    • Performance Optimized — Efficient resource usage and caching

    Developer Friendly

    • Plugin System — Extend without modifying core code
    • Theme Support — Complete visual customization

    Feature Rich

    • User Management
      • Registration, Onboarding, Restriction, Wiping, Moderation, PW Recovery all in AP
    • Leaderboards
      • Global and country-specific rankings
      • Clan Rankings
    • Statistics — Detailed player stats and history
    • Much more — Cause of the Plugin System
